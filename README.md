## Contexte

Implémentation du "*barbershop problem*" en c++, pour la remise de la ressource R5.A.05.

Afin de rendre cette situation plus concrète, nous avons choisi de créer un scénario où un opérateur communique avec des clients par téléphone en utilisant un système de file d'attente qui peut être pleine.

## Exécution

- CMake : compiler le fichier main.cpp à la racine du projet.
- QMake : compiler le fichier main.cpp dans le dossier Barbershop_problem (pour QTCreator)
